## 1. 경영진 요약 (Executive Summary)

### 1.1 제안 요약

**SDC Customer Quality System**은 AI 기술과 빅데이터 분석을 결합한 **차세대 디스플레이 품질관리 통합 솔루션**입니다.

본 시스템은 고객 불만 접수부터 원인 분석, 대책 수립, 고객 회신까지 **전체 CS 워크플로우를 자동화**하고, **NVIDIA Cosmos Reason 7B VLM**을 활용한 AI 기반 결함 분석, **Neo4j 지식그래프**를 통한 유사 사례 검색, **Apache Spark** 기반 빅데이터 분석 기능을 제공합니다.

### 1.3 주요 기능

| 구분 | 기능 | 설명 |
|------|------|------|
| **CS 워크플로우** | 4단계 프로세스 | 불만접수 → 1차분석 → 2차분석 → 보고서생성 |
| **결재 시스템** | 다단계 승인 | 팀장 → 부장 → 상무 결재 라인, 기한 관리 |
| **VLM 분석** | AI 결함 분석 | Cosmos Reason 7B 기반 이미지 분석 |
| **GraphRAG** | 지식그래프 | Neo4j 기반 유사 사례 검색 및 원인 추적 |
| **빅데이터** | 패턴 분석 | Spark 기반 대용량 품질 데이터 분석 |
| **대시보드** | 실시간 현황 | 품질 KPI 모니터링 및 시각화 |

### 3.2 기능 범위

#### 3.2.1 핵심 기능 (In Scope)

| 대분류 | 중분류 | 기능 |
|--------|--------|------|
| **CS 워크플로우** | Step 1. 불만 접수 | 고객 정보 등록, 결함 정보 입력, 이메일 생성 |
| | Step 2. 1차 분석 | 초기 분석, 미팅 관리, 업무 할당 |
| | Step 3. 2차 분석 | 상세 원인 분석, 대책 수립 |
| | Step 4. 보고서 | 최종 보고서 생성, 고객 회신 |
| **결재 시스템** | 결재 관리 | 결재 요청/승인/반려, 다단계 결재 |
| | 기한 관리 | 기한 추적, 알림 발송 |
| **VLM 분석** | 이미지 분석 | 결함 유형 분류, 위치 탐지, 심각도 평가 |
| **GraphRAG** | 지식그래프 | 노드/관계 관리, 유사 사례 검색 |
| **빅데이터** | 분석 파이프라인 | 데이터 수집, ETL, 패턴 분석 |
| **대시보드** | 시각화 | 품질 현황, KPI, 차트 |

#### 3.2.2 제외 범위 (Out of Scope)

| 항목 | 사유 |
|------|------|
| ERP 연동 | 2차 사업에서 검토 |
| MES 실시간 연동 | 별도 인터페이스 프로젝트로 진행 |
| 외부 고객 포털 | 고객사별 협의 후 추가 개발 |
| 모바일 앱 | 웹 반응형으로 대체 |

#### 5.2.1 불만 접수 단계

**문제점:**

| 표준 양식 부재 | 접수 품질 불균일 |

**현행 방식:**
- 품질 담당자 경험 기반 분석

- 현미경/측정 장비로 수동 검사

**소요 시간:** 평균 2-3일

**문제점:**
| 문제 | 영향 |
|------|------|
| 경험 의존 | 담당자별 분석 품질 차이 |
| 사례 검색 어려움 | 유사 사례 활용 미흡 |
| 수동 분석 | 시간 소요, 휴먼 에러 |
| 지식 휘발 | 퇴직 시 노하우 유실 |

#### 5.2.3 보고서 작성 단계

**현행 방식:**
- Word로 수동 작성
- 과거 보고서 복사 후 수정

**소요 시간:** 평균 4시간

**문제점:**
| 문제 | 영향 |

## 부록 A. 용어 정의

| 용어 | 정의 |
|------|------|
| **VLM** | Vision Language Model. 이미지와 텍스트를 함께 이해하고 처리할 수 있는 AI 모델 |
| **Cosmos Reason** | NVIDIA에서 개발한 비전-언어 모델로, 이미지 분석 및 추론 능력이 우수 |
| **LoRA** | Low-Rank Adaptation. 대규모 언어 모델을 효율적으로 파인튜닝하는 기법 |
| **GraphRAG** | Graph Retrieval-Augmented Generation. 지식그래프를 활용한 검색 증강 생성 |
| **Neo4j** | 그래프 데이터베이스 관리 시스템 |
| **Apache Spark** | 대규모 데이터 처리를 위한 분산 컴퓨팅 프레임워크 |
| **Gradio** | Python 기반 웹 인터페이스 구축 프레임워크 |
| **CS** | Customer Service. 고객 서비스, 본 문서에서는 고객 불만 처리를 의미 |
| **MES** | Manufacturing Execution System. 제조 실행 시스템 |
| **UAT** | User Acceptance Test. 사용자 인수 테스트 |
| **M/M** | Man-Month. 1명이 1개월 동안 수행하는 작업량의 단위 |
| **ROI** | Return on Investment. 투자 대비 수익률 |
| **BEP** | Break-Even Point. 손익분기점 |
| **Chain-of-Thought** | 단계별 추론 과정을 명시적으로 수행하는 AI 추론 기법 |
