# 디스플레이 결함 분석 시스템 - 발표 자료 개요

---

## Slide 1: 표지

**디스플레이 결함 분석 시스템**
*Cosmos Reason VLM + GraphRAG 기반 지능형 품질 검사*

---

## Slide 2: 문제 정의

### 현재 과제
- 기존 AOI: 결함 탐지만 가능, **"왜?"에 대한 답 없음**
- 엔지니어 의존: 근본원인 분석에 전문가 필요
- 지식 단절: 경험 축적 및 전파 어려움

### 해결책
- VLM으로 결함 탐지 + 추론 자동화
- GraphRAG로 지식 체계화 및 재활용

---

## Slide 3: 시스템 개요

```
┌──────────────────────────────────────────────────┐
│                                                  │
│    [이미지] → [VLM 분석] → [GraphRAG] → [결과]    │
│               <think>      지식그래프             │
│               <answer>     근본원인 추론          │
│                                                  │
└──────────────────────────────────────────────────┘
```

---

## Slide 4: 핵심 기술 - VLM

### Cosmos Reason 7B
- NVIDIA 제공 추론 특화 비전-언어 모델
- **`<think>` 태그**: 추론 과정 명시화
- **`<answer>` 태그**: 구조화된 결과 출력

### LoRA Fine-tuning
- 전체 8.4B 중 **2.24%만 학습**
- 빠른 도메인 특화 (10 에폭, ~80분)
- 손실: 13.17 → 5.21 (60% 감소)

---

## Slide 5: 핵심 기술 - GraphRAG

### 온톨로지 구조
```
[결함] ──CAUSED_BY──> [근본원인]
   │                      │
OCCURS_IN              OCCURS_IN
   │                      │
   ▼                      ▼
[공정] ────USES────> [설비/자재]
```

### 활용
- 결함-원인 관계 학습
- 공정 연관성 추론
- 지식 축적 및 재활용

## Slide 6: 지원 결함 유형

| 유형 | 설명 | 예시 원인 |

| 이물질 | 오염/파티클 | 클린룸 환경 |

## Slide 8: 분석 결과 예시

### 입력
디스플레이 패널 검사 이미지

### 출력
```
<think>
이미지를 분석한 결과, 좌측 상단 영역에서
라인 결함 패턴이 관찰됩니다. 결함의 형태와
분포를 고려할 때, 전극 터치 패턴과의
상호작용이 원인으로 추정됩니다.
</think>

<answer>
결함 유형: 라인 결함
위치: 좌측 상단
심각도: HIGH
가능한 원인: 전극 터치 패턴과의 상호작용
권장 조치: 균일한 전극 패턴 확인
</answer>
```
